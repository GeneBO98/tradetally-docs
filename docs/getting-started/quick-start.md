# Quick Start

For most self-hosted installs, skip the native/source setup and start with Docker.

## Recommended Path

- **Docker self-hosting**: [Docker Deployment](../deployment/docker.md)
- **Source install for local development**: [Native Installation](installation.md)

## After You Install

1. Open TradeTally in your browser.
2. Create the first user account. The first user becomes the admin.
3. Configure API keys if you want market data, broker sync, or AI features.
4. Import your first trades from the **Import** screen.

## Notes

- Docker is the primary self-hosted path and is what the `Quick Start` navigation now points to.
- The native install guide exists for developers and advanced custom environments.
